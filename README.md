# classexample

A demo repository for learning Git — used in a class presentation.

## What this repo covers
- Cloning a repo with Git Bash and VS Code
- Creating branches
- Making commits
- Pushing to GitHub
- Merge conflicts

## Files
| File | Purpose |
|------|---------|
| `calculator.py` | Simple Python calculator — edit this to practice |
| `notes.txt` | Class notes — add your name to practice commits |

## How to get started
```bash
git clone https://github.com/Hparker2713/classexample.git
cd classexample
git checkout -b feature/your-name
```
